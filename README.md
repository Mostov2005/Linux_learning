# Linux Learning

Репозиторий содержит мои решения задач по изучению Linux.
Все работы выполнялись в учебных целях для практического освоения различных аспектов системы.

## Изученные темы и папки

-  **Work with files** – работа с файлами: создание, копирование, перемещение, удаление.

-  **User manage** – управление пользователями и правами доступа.

-  **File systems** – работа с файловыми системами и дисками.

-  **Bash** – написание скриптов для автоматизации задач.

-  **Systemd** – работа с сервисами и демонами Linux.

-  **SSH** – подключение к серверу через SSH.

-  **Samba** – настройка и запуск сервера Samba, работа с общими папками.

-  **Firewall** – настройка правил брандмауэра и безопасности сети.

---

Все решения направлены на практическое освоение Linux и системного администрирования.
